# dmap
疾管署 - 疫情與輿情交叉分析比對


## 安裝步驟

1. 安裝git
- git clone https://github.com/ywchiu/dmap.git

2. 安裝nginx 
-  sudo apt-get install nginx


3. 將dmap 移到 /webapps
- sudo mkdir /webapps
- sudo mv dmap /webapps/


4. 安裝PostgreSQL
- sudo apt-get update
- sudo apt-get install postgresql postgresql-contrib
